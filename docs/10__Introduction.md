# Introduction

AliDocs Bundle is a basic tool for creating sitemaps in a symfony project
The purpose is to quickly publish sitemaps with simple configuration

Basically, a sitemap is a list of nodes configured in the yaml configuration of the bundle

## Features

- Create as sitemaps as needed by edition the bundle config file

- Create url nodes in sidemap, with route, dynamic route with Entity and Repository config

- Easy overwrite design and template

- No database requested
